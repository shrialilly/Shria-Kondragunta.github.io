Final Project

Description

This is a game made using scratch. The main player is Mario. The game name is Mario Run. In this game Mario has to dodge many things. This game has 3 levels and has many scenes. In the first level Mario has to dodge pink sqaures and he can do this when the player presses the SPACEBAR. In the second level Mario has to dodge the purple birds and to do this the player has to hit the down arrow key. In level 3 Mario has to dodge triangles and do this the player has to hit SPACEBAR. To pass each level Mario has to survive for a certain amount of time.
